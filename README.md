# 🎨 ペイントアプリ – 純粋な HTML + JavaScript で動くブラウザキャンバス  

サーバー不要、**完全にブラウザ上で動作するペイントアプリ**です。  
Vanilla JavaScript と Tailwind CSS を使い、シンプルかつモダンな UI に仕上げています。

> 💡 主要なデスクトップブラウザー（Chrome, Edge, Firefox, Safari）ならすぐに動きます。ビルドは不要です。

---

## 🚀 主な機能

| 機能 | 内容 |
|------|------|
| **フリーハンド描画** | ペンツールで直線をリアルタイムに描けます。 |
| **図形ツール** | • 円<br>• 四角<br>• 三角<br>• 多角形（クリックで頂点追加、ダブルクリックで確定）<br>• 矢印（ドラッグで直線＋矢印頭部を描画） |
| **文字入力** | クリックするとプロンプトが表示され、テキストをキャンバスに書き込みます。 |
| **色選択** | スタイルシートと同じカラーピッカーで好きな色を設定できます。 |
| **Undo / Redo** | 無限履歴（すべての描画操作が復元可能）。 |
| **全消し** | クリック一つでキャンバス上のすべてを消去します。 |
| **クリップボードにコピー** | 現在のキャンバスを PNG 画像としてシステムクリップボードへコピーできます。 |
| **レスポンシブ設計** | キャンバスはウィンドウサイズに合わせて自動調整されます。UI は中央に固定。 |
| **Tailwind CSS スタイリング** | シンプルで洗練された UI、青緑グラデーションテーマ。 |

## 📦 取得方法

インストール不要です。リポジトリをクローンまたは ZIP をダウンロードして `index.html` を開くだけ。

```bash
# リポジトリのクローン（例）
git clone https://github.com/<your-username>/paint-app.git
cd paint-app

# ブラウザで index.html を開く
open index.html      # macOS
start index.html     # Windows
xdg-open index.html # Linux
```

> **クリップボード機能を確実に動かしたい場合**  
> (例：ローカルサーバーが必要になることがあります)  
> ```bash
> python3 -m http.server 8000
> # その後、http://localhost:8000 にアクセス
> ```

## 📖 使い方

1. **ツールを選択**（ペン・円・四角・三角・矢印・多角形・文字）。  
2. **色を決める** カラーピッカーで好きな色に。  
3. **キャンバス上で描画**  
   * 形状以外はドラッグでサイズ確定。  
   * 多角形はクリックで頂点追加、ダブルクリックまたは「全消し」で確定。  
4. **Undo / Redo ボタン**で操作履歴を戻したり進めたり。  
5. **コピー** ボタンで現在のキャンバスを PNG 画像としてクリップボードへ。  
6. **全消し** ボタンで一括削除。

## 🛠️ 開発

このプロジェクトは単一ページアプリです。ビルドツールやパッケージマネージャーは不要です。  
貢献したい方は以下を参考にしてください：

1. このリポジトリをフォーク＆クローン。  
2. `index.html` / `script.js` を編集。  
3. ブラウザでローカル起動（上記方法）またはローカルサーバーで確認。  
4. コミット → プッシュ → Pull Request。
